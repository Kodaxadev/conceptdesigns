# Kodaxa Concept Designs

Experimental design space for Kodaxa portfolio concepts.

This repo is intentionally separate from the production portfolio. It is where visual systems, Blender studies, motion references, and web interaction prototypes can be tested before anything graduates to `kodaxa.dev`.

## First Concept

- Split-flap mechanical board study
- Blender 5.1.2 prototype renders
- Static Astro concept page
- Future target: web-native flip-board interaction

## Commands

```bash
npm install
npm run check
npm run build
npm run dev
```
